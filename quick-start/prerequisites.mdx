## Required Software

### üê≥ Container Platform

<Warning>
  Docker is the primary deployment method. Ensure you have the latest stable version installed.
</Warning>

<Steps>
  <Step title="Install Docker">
    ```bash
    # Ubuntu/Debian
    curl -fsSL https://get.docker.com -o get-docker.sh
    sudo sh get-docker.sh
    sudo usermod -aG docker $USER

    # macOS
    brew install --cask docker

    # Or download from https://docker.com
    ```
  </Step>
  <Step title="Install Docker Compose">
    ```bash
    # Linux
    sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
    sudo chmod +x /usr/local/bin/docker-compose

    # macOS (included with Docker Desktop)
    # Windows (included with Docker Desktop)
    ```
  </Step>
  <Step title="Verify Installation">
    ```bash
    docker --version
    docker-compose --version

    # Should output version numbers
    # Docker version 24.0.0 or later
    # Docker Compose version 2.0.0 or later
    ```
  </Step>
</Steps>

#### Installation Commands

<Tabs>
  <Tab title="Node.js & pnpm">
    ```bash

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    ```

## Environment Preparation

### üîß System Configuration

<Tabs>
  <Tab title="Linux Optimization">
    ```bash
    # Increase file descriptor limits
    echo "* soft nofile 65536" | sudo tee -a /etc/security/limits.conf
    echo "* hard nofile 65536" | sudo tee -a /etc/security/limits.conf

    # Configure Docker daemon
    sudo mkdir -p /etc/docker
    sudo tee /etc/docker/daemon.json <<EOF
    {
      "log-driver": "json-file",
      "log-opts": {
        "max-size": "10m",
        "max-file": "3"
      },
      "storage-driver": "overlay2"
    }
    EOF

    sudo systemctl restart docker
    ```
  </Tab>
  <Tab title="macOS Configuration">
    ```bash
    # Increase Docker memory allocation
    # Docker Desktop > Settings > Resources > Memory: 8GB+

    # Install development dependencies
    xcode-select --install

    # Configure shell (if using zsh)
    echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
</Tabs>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Docker Issues">
    **Problem**: Permission denied when running Docker commands

    **Solution**:
    ```bash
    sudo usermod -aG docker $USER
    newgrp docker
    # Or restart your terminal
    ```
  </Accordion>

  <Accordion title="Port Conflicts">
    **Problem**: Port already in use errors

    **Solution**:
    ```bash
    # Find process using port
    sudo lsof -i :8080

    # Kill process if safe to do so
    sudo kill -9 PID
    ```
  </Accordion>

  <Accordion title="Memory Issues">
    **Problem**: Out of memory errors during Docker builds

    **Solution**:
    - Increase Docker Desktop memory allocation (macOS/Windows)
    - Add swap space (Linux)
    - Use multi-stage builds to reduce memory usage
  </Accordion>
</AccordionGroup>
